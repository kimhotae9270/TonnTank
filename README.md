
# 🎮 ToonTanks (UE C++)

탱크를 조종해 포탑을 파괴하는 미니 슈팅 게임. 마우스로 조준, 키보드로 이동하며 포탄과 이펙트/사운드/카메라 셰이크까지 갖춘 **언리얼 C++ 학습용 프로젝트**입니다.

> UE 5.4 / C++ / 컴포넌트 지향 / 간단한 게임 루프

---

## ✨ 주요 특징

- **플레이어 탱크 조작**: 전진/회전, 마우스 커서 지점으로 터렛 회전, 발사  
- **적 포탑 AI**: 사거리 내 플레이어 감지 → 터렛 추적 → 주기적 사격  
- **프로젝트타일 시스템**: 피격 시 데미지 적용, 파티클/사운드/카메라 셰이크, 소유자/피격 예외 처리  
- **체력/사망 처리**: 데미지 수신, 체력 0 시 게임 모드에 사망 알림  
- **게임 흐름**: 스타트 딜레이 후 입력 활성화, 모든 포탑 격파 시 승리 / 탱크 파괴 시 패배

---

## 🧩 클래스 구조 개요
```markdown
ABasePawn - 공통 베이스: 캡슐/베이스/터렛 메시, 발사 지점, 터렛 회전 & 발사
├─ ATank - 플레이어: 카메라 붐/카메라, 입력(이동/회전/발사), 커서 히트로 조준
└─ ATower - AI 포탑: 사거리 내 탱크 추적, 타이머 기반 발사 조건 체크

AProjectile - 투사체: 이동 컴포넌트, OnHit에서 데미지/이펙트/사운드/셰이크, 자멸

UHealthComponent - 체력 컴포넌트: 데미지 바인딩, 0 이하 시 GameMode에 ActorDied 전달

AToonTankksGameMode - 게임 진행: 스타트/게임오버, 타깃 타워 수 관리, 입력 활성화 지연

AToonTanksPlayerController - 입력 토글/마우스 커서 표시 제어
```
## ⌨️ 조작법 (기본)

| 동작 | 키/마우스 |
|---|---|
| 전진/후진 | W / S (또는 축 매핑 `MoveForward`) |
| 좌/우 회전 | A / D (또는 축 매핑 `Turn`) |
| 발사 | 마우스 좌클릭 (액션 `Fire`) |
| 조준 | 마우스 커서 위치로 터렛 자동 조준 |

> 프로젝트 설정 → Input에서 `MoveForward`, `Turn`, `Fire` 매핑을 확인/수정하세요.
